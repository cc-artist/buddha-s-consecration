## 问题分析
前端网页无法预览的原因可能是：
1. **依赖包未安装**：前端目录中缺少`node_modules`文件夹
2. **开发服务器未启动**：需要运行Next.js开发服务器

## 解决方案

### 步骤1：安装前端依赖
在`cyber-buddha-blessing`目录中执行：
```bash
npm install
```

### 步骤2：启动开发服务器
安装依赖后，执行：
```bash
npm run dev
```

### 步骤3：验证预览
开发服务器启动后，会在控制台显示访问URL（通常是`http://localhost:3000`），通过浏览器访问该URL即可预览前端网页。

## 技术说明
- 项目使用Next.js 16.0.1版本
- 开发服务器默认运行在3000端口
- 依赖包安装完成后会生成`node_modules`目录

此方案通过安装必要的依赖并启动开发服务器，解决前端网页无法预览的问题。
